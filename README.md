# Ecode-Tabs

> All kinds of tabs about js(dev and master branch)

## Build Setup

``` bash
# 数据库
将server目录下的tabs.sql导入到数据库

# install dependencies
npm install

# 服务端
命令行进入server目录，执行node index.js

# serve with hot reload at localhost:2000
npm run dev
```
